## 1. Operator 

Pada bab khusus, kami akan mencatat seluruh hal-hal yang tidak ada pada repositori asli dan juga kami menyandarkan catatan ini pada eksperimen yang telah kami lakukan. 

A. Operator aritmatika 

Dalam progamming itu tidak akan pernah lepas dari yang namanya matematika, at least di progamming itu bisa melakukan perhitungan sederhana seperti penjumlahan, pengurangan, pembagian dan perkalian. Di Banjarscript juga terdapat juga. 

contoh 

```
wadah angkaPertama = 10; 
wadah angkaKedua = 2; 

konstanta penjumlahan = angkaPertama + angkaKedua; 
konstanta pengurangan = angkaPertama - angkaKedua; 
konstanta perkalian = angkaPertama * angkaKedua; 
konstanta pembagian = angkaPertama / angkaKedua;

tampaiakan(penjumlahan);
tampaiakan(pengurangan); 
tampaiakan(perkalian); 
tampaiakan(pembagian); 
```

B. Operator pembanding

Operator pembanding digunakan untuk membandingkan dua nilai. Yang dimana hasil yang dimunculkan ialah `true` atau `false`. Namun dalam Banjarscript mereka dideklarasikan sebagai `bujur` dan `benar` untuk `true` dan `salah` adalah `false`

| **Operator** | **Nama Operator**            | **Contoh**  | **Hasil** | **Keterangan**                                             |
| ------------ | ---------------------------- | ----------- | --------- | ---------------------------------------------------------- |
| `==`         | Sama dengan                  | `5 == "5"`  | `true`    | Membandingkan nilai saja (tanpa lihat tipe data)           |
| `===`        | Sama dengan (strict)         | `5 === "5"` | `false`   | Membandingkan nilai **dan** tipe data                      |
| `!=`         | Tidak sama dengan            | `5 != "5"`  | `false`   | Hanya membandingkan nilai                                  |
| `!==`        | Tidak sama dengan (strict)   | `5 !== "5"` | `true`    | Membandingkan nilai **dan** tipe data                      |
| `>`          | Lebih besar dari             | `10 > 5`    | `true`    | Bernilai true jika kiri lebih besar dari kanan             |
| `<`          | Lebih kecil dari             | `10 < 5`    | `false`   | Bernilai true jika kiri lebih kecil dari kanan             |
| `>=`         | Lebih besar atau sama dengan | `10 >= 10`  | `true`    | Bernilai true jika kiri lebih besar atau sama dengan kanan |
| `<=`         | Lebih kecil atau sama dengan | `10 <= 5`   | `false`   | Bernilai true jika kiri lebih kecil atau sama dengan kanan |

```
// operator pembanding

wadah a = 10;
wadah b = 5;

// Sama dengan (==)
tampaiakan(a == b);   // false

// Tidak sama dengan (!=)
tampaiakan(a != b);   // true

// Sama dengan dan tipe datanya juga sama (===)
tampaiakan(a === 10); // true

// Tidak sama dengan dan tipe datanya juga berbeda (!==)
tampaiakan(a !== "10"); // true

// Lebih besar dari (>)
tampaiakan(a > b);    // true

// Lebih kecil dari (<)
tampaiakan(a < b);    // false

// Lebih besar atau sama dengan (>=)
tampaiakan(a >= 10);  // true

// Lebih kecil atau sama dengan (<=)
tampaiakan(a <= 5);   // false
```
