## 5. return 

`return` merupakan hasil akhir dari `function` jika fungsi tidak di isi maka `return` tidak akan memberikan hasil apa-apa/undefined. 

Dalam BanjarScript, `return` didefinisikan sebagai `balikan` atau `bulikakan`. 

Contoh : 
```bjs
fungsi tambah(c, d){
    balikan c + d;
};

wadah meme = tambah(4, 9)
tampaiakan(meme);
```
