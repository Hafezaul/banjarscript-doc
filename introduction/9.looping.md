## 7. Looping 

Looping digunakan untuk melakukan perulangan sesuai program yang kita buat. Jenis perulangan ada tiga, yaitu : 

1. `for` 
2. `while`
3. `do-while`

Ketiga kode tersebut memiliki kegunaannya masing-masing. Namun, didalam Banjarscript hanya ada `for` yang dideklarasikan sebagai `gasan` dan `while` yang dideklarisikan sebagai `munkada`, `selagi`, `selama`

contoh : 

```
// for looping
gasan (wadah i = 1; i <= 5; i++) {
  tampaiakan("Perulangan ke-" + i);
}

// while

wadah i = 1;
munkada (i <= 5) {
  tampaiakan("Looping while ke-" + i);
  i++;
}
```
