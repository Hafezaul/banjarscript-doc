## 6. if condition

`if` merupakan program yang dapat mengeksekusi 1 program dari 2 kemungkinan dengan syarat terpenuhi. 

contoh dalam javascript 

```js
let umur = 17; 

if (umur > 18) {
    console.log("Umur anda sudah sesuai");
} else {
    console.log("Masih bocil amat");
}
```
Contoh di atas merupakan penggunaan if dalam Javascript, tetapi jika diterapkan pada BanjarScript maka kita bisa menggunakan `mun` untuk `if` dan `nanglain` atau `lain` sebagai `else`

Contoh 

```bjs
wadah umur = 17; 

mun (umur > 18) { 
    tampaiakan("Umur anda sudah dewasa");
} nanglain {
    tampaiakan("Masih bocil yeee");
}
```

Di BanjarScript penggunaan if condition hanya terbatas pada `if` dan `else` saja untuk `if-else-else` atau `switch` belum dikembangkan. Mohon untuk selalu update terkait bahasa ini.
