## 4. Fungsi
Fungsi pada BanjarScript menggunakan `fungsi` sebagai code yang dieksekusi, fungsi pada BanjarScript juga tidak terlalu jauh dari Javascript. Namun, ia disesuaikan dengan bahasa Banjar. 

Contoh :
```bjs 
fungsi kali(a, b){
    hasilKali = a * b;
    tampaiakan("Hasil dari ini = " + hasilKali);
}

kali(2, 3);
```
